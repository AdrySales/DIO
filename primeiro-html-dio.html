Inspecionador de Código

Inspecionador de elemento é uma ferramenta que permite visualizar o código-fonte da página web escolhida. Além de visualização de código é possível fazer alterações temporárias e visualizar como ficam sem a necessidade de mexer no código fonte.


Estrutura básica do HTML

O documento HTML sempre inicia com a estrutura básica. Esta estrutura é quase fixa,o atalho para ela no html é "html:5"Após dar enter a estrutura aparece.. A estrutura em sí e os elementos que compõe o html são compostos por tags. As tags são usadas para o navegador conseguir identificar a estrutura do site, existem várias tags para diferentes objetivos, cada tag possui seus atributos que podem ser específicos daquela tag ou não.

Atributos HTML são palavras especiais usadas dentro da tag de abertura para controlar o comportamento do elemento. Os atributos HTML são um modificador de um tipo de elemento HTML. Com eles podemos identificar melhor um elemento, informar qual arquivo aquela tag deve utilizar, indicar o tipo de um campo de texto.Fonte:www.alura.com.br

Tags HTML

->Tags h1 até h6

Os elementos HTML <h1> até <h6> representam seis níveis de título de seção. <h1> é o nível de seção mais alto e <h6> é o mais baixo.

Modo de uso: <h1> Seu texto aqui </h1>


->Tag P

O elemento <p> representa um parágrafo, são blocos de texto identados.

Modo de uso: <p> Seu texto aqui </p>


->Tag Mark

O elemento <mark> faz com que o texto fique destacado, assim como se fosse um marcador de texto.

Modo de uso: <mark> Seu texto aqui </mark>


->Tag Small

O elemento <small> deixa o texto em um tamanho reduzido.

Modo de uso: <small> Seu texto aqui </small>


->Tag I

O elemento <i> faz com que o texto fique com a fonte em italic.

Modo de uso: <i> Seu texto aqui </i>


->Tag U

O elemento <u> deixa o texto sublinhado.

Modo de uso: <u> Seu texto aqui </u>


->Tag Strong

O elemento <strong> deixa o texto em negrito.

Modo de uso: <strong> Seu texto aqui </strong>


->Listas ordenadas e não ordenadas

Para fazer listas ordenadas ou não, são utilizadas 3 divs diferentes. A div <li> participa da construção dos dois tipos de lista. A lista ordenada é constituida pela div <ol> e a não ordenada pela div <ul>. É possível também utilizar estruturas ordenadas e não ordenadas embutidas, formando listas mais dinâmicas e diversas de acordo com a necessidade.

Modo de uso:

<ol>

<li>
<p> Elemento 1 </p>
</li>

<li>
<p> Elemento 2 </p>
</li>

<\ol>

__________________

<ul>

<li>
<p> Elemento 1 </p>
</li>

<li>
<p> Elemento 2 </p>
</li>

<\ul>


->Tag A

A tag <a> é utilizada para hyperlinks, é possível referenciar links de páginas externas como também páginas do próprio website além de fazer marcações. No atributo href é colocado o link em questão.

Modo de uso: <a href="#"> Seu link aqui </a>


->Tag HR

O elemento <hr> desenha uma linha reta na página. Pode ser usada para várias finalidades.

Modo de uso: <hr>


->Tag Sub

O elemento <sub> define um texto subscrito. O texto com essa tag fica menor que o normal e um pouco abaixo do elemento anterior a ele.

Modo de uso: <sub>Seu texto aqui.</sub>


->Tag Sup

O elemento <sup> define um texto subrescrito. O texto com essa tag fica menor que o normal e um pouco acima do elemento anterior a ele.

Modo de uso: <sup>Seu texto aqui.</sup>


->Tag Blockquote

O elemento <blockquote> desenha uma linha reta na página. Pode ser usada para várias finalidades.

Modo de uso: <blockquote> Seu texto aqui.</blockquote>


->Tag Font

O elemento <font> A tag de fonte é usada para alterar a cor, o tamanho e o estilo de um texto que se encontra dentro dela.

Modo de uso: <font size="5"> Seu texto aqui.</font>


->Tag Del

O elemento <del> representa uma parte do texto que foi excluída de um documento.

Modo de uso: <del> Seu texto aqui.</del>


->Tag img

O elemento <img> é utilizado para colocar uma imagem na página.

Modo de uso: <img> src="linkDaImagem.jpg" />
